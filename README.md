Atlys Assignment APP
=======
### Running Steps:
  - ```git clone https://github.com/mk2683/atlys-app.git```
  - ```nvm use 20```
  - ```npm i```
  - ```npm start```

### Dependencies
  - React
  - Typescript
  - tailwind
